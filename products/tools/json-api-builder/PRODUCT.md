# json-api-builder

Spin up a CRUD JSON API server from a JSON config.

json-api-builder is a lightweight CLI tool for developers who need a backend **now** — for prototyping, dashboards, mobile apps, internal tools, or frontend development — without setting up a full backend stack.

---

## Who this is for
- Frontend developers
- Indie builders
- Product designers prototyping flows
- Teams building internal tools
- Anyone who needs a fast, disposable API

---

## What it does
- Reads a JSON config file
- Starts a Fastify server
- Automatically creates CRUD endpoints per resource
- Applies basic validation from config
- Optionally persists data to disk

---

## Key features
- Zero setup (Node.js only)
- CLI-based workflow
- Predictable REST endpoints
- Optional file persistence
- Human-readable config
- No external database required

---

## What’s included
- CLI tool (`init`, `dev`)
- JSON config format
- CRUD API generation
- In-memory and file-backed storage
- Examples and documentation

---

## What’s intentionally excluded (v0.x)
- Authentication
- Authorization / roles
- Relations or joins
- Pagination or sorting
- Hosted dashboard or UI

These are planned for future versions.

---

## Typical use cases
- Frontend development without a backend
- Rapid MVP prototyping
- Mock APIs for demos
- Internal admin tools
- Teaching / learning REST APIs

---

## Tech stack
- Node.js
- TypeScript
- Fastify
